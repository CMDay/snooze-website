# CMDaySnooze

Promo website voor de CMDay.snooze app. Gebouwd met [Bootstrap](http://getbootstrap.com/).

## Benodigdheden

* Node.js
* Grunt.js

## Ontwikkelen

In de console / terminal:

	$ npm install
	$ grunt
